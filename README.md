## Features 特性

一个Vscode插件，让您可以把任意位置上的文件推送到您的Github-page站点。

## Requirements 依赖

如果您的电脑上装有`git`，那么它不需要任何依赖。

## Let's Start 快速开始  

### 三个命令    
`NEWPAGE-PUSH:PUSH 推送` 向您推送您的文档     
`NEWPAGE-PUSH:INIT 初始化` 在默认位置初始化站点的git仓库，便于管理     
`NEWPAGE-PUSH:REFRESH 刷新` 同步您的站点
